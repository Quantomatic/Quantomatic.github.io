How to Contribute...

GitHub Project » Google Group »

You can help by running QuantoDerive sending bug reports to Aleks or (preferably) by reporting them as issues.

If you would like to contribute to the Quantomatic project, get in touch with Aleks or fork us on GitHub.

Some documentation for developers is on the project wiki. You'll also want to join the Quantomatic Google group.
